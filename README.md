# Creating Cohorts of Songs (Spotify - Rolling Stones)
This project performs exploratory data analysis and clustering on Rolling Stones songs using Spotify's audio features.

## Key Tasks
- Data cleaning and feature selection  
- Exploratory Data Analysis (EDA)  
- Dimensionality Reduction using PCA  
- K-Means Clustering (k=3)  
- Cohort interpretation and recommendations  

## Key Insights
- Songs grouped into three clusters: energetic rock, mellow acoustic, and upbeat rhythmic hits.
- Top albums by popularity: Sticky Fingers and Some Girls.
- Energy and Loudness strongly correlate (r=0.7).

## Technologies Used
Python, Pandas, Matplotlib, Seaborn, Scikit-learn
